TL;DR

  1. STAGE 페이지 구조를 S0~S9 9섹션으로 재구성. 신규 개념 "첫인상 진단"이 히어로 CTA·Basic 첫 요소·마지막 CTA로 관통.
  2. Basic/Premium을 각각 5요소·4요소로 재정의. Basic 부제는 유지, Premium은 숏폼→월간 콘텐츠 운영으로 명칭 변경.
  3. 블로그 팩 3종(55/45/80만) + 단건 3티어(15/20/25만) 신설. 폼 스키마 blog_build 단일 → 5개로 확장 필요.

1-3. 핵심 카피 갱신

S0 · Hero 헤드
"홍보물은 누구에게나 쉬워졌습니다. 근데, 어디에 어떻게 쓰고 계신가요?"
10to10 STAGE는 학원 홍보물을 채널별로 다시 쓰기 좋게 설계하고, 학부모가 검색하고 비교하고 문의하기까지의 흐름을 정리합니다.
CTA: 패키지 보기 · 첫인상 진단 문의하기 ← 신규
S2 · 왜 10to10인가
"제작보다 중요한 건 배치, 배치보다 중요한 건 연결입니다."
홍보물은 많아졌지만, 문의로 이어지는 구조는 여전히 부족합니다.
S9 · 마지막 CTA
"우리 학원은 지금, 어떻게 보이고 있을까요?"
플레이스·블로그·인스타·검색 결과까지. 학부모가 학원을 처음 만나는 화면부터 점검하고 개선 방향을 제안드립니다.

1-4. Basic / Premium 재정의

Basic

학원 운영을 받쳐주는 기본 마케팅 패키지
  • 첫인상 진단 리포트플레이스·블로그·인스타·검색 결과 노출 분석 + 지역 경쟁 학원 비교 + 실행안
  • 기본 홍보 자산 세트학원 소개 · 강사 · 커리큘럼 · 성과·후기 사례
  • 플레이스 첫인상 세팅대표 이미지 · 소개글 · 소식 배너 · 적용 목업
  • 시즌 홍보 기본 세트중간·기말 / 방학특강 / 개강 / 설명회
  • 10to10 연계 안내설명회 · 레벨테스트 · 학부모 문의 연결 안내
⚠ 갭 · 현재 월간 운영 리포트 요소가 노션안엔 없음 → Premium 이관 여부 확인 필요 (Q1)

Premium BEST

디자인·콘텐츠·광고까지 전담 매니저 1:1 케어
  • Basic 전체 포함5요소 그대로
  • 월간 콘텐츠 운영릴스·숏츠 · 카드뉴스 · 소식글 · 월간 콘텐츠 캘린더
  • 디지털 광고 운영네이버·구글 키워드 광고 · 소재 기획 · 성과 리뷰
  • 운영 관리월 1회 리뷰 미팅 · 긴급 대응 · 시즌 조율 · 콘텐츠 수정 방향
변경 · 숏폼 콘텐츠 → 월간 콘텐츠 운영으로 확장, 전담 매니저 → 운영 관리로 이름 정리, 광고비 별도 안내 문구 명시.
