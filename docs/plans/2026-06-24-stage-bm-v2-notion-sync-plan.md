## 1. 노션 개선안 핵심 요약

### 1-1. 블로그 팩 재정의 (Options 섹션 → 별도 카드셋)

| 상품 | 가격 | 발행량 | 포지션 |
|---|---|---|---|
| **블로그 첫달 세팅팩** | 55만원 (1회성) | 첫 달 4건 + 세팅 | ★★★★★ |
| **월간 4건 운영팩** | 45만원/월 | 월 4건 | ★★★★★ |
| **월간 8건 성장팩** | 80만원/월 | 월 8건 | ★★★★☆ |
| 일반 단건 | 15만원 | 1건 | 학원 소식·정보·짧은 홍보 |
| 상세 단건 | 20만원 | 1건 | 방학특강 상세·커리큘럼 |
| 고관여 단건 | 25만원 | 1건 | 원장/강사 인터뷰·설명회 상세 |

**운영 룰**
- 첫 달은 고정 구성, 이후는 월 4건 단위 개별 요청형
- 단건은 10만원 이하로 내리지 않음 (월간팩 잠식 방지)

### 1-3. 핵심 카피 갱신
- **히어로 헤드**: "홍보물은 누구에게나 쉬워졌습니다. 근데, 어디에 어떻게 쓰고 계신가요?"
- **히어로 서브**: "10to10 STAGE는 학원 홍보물을 채널별로 다시 쓰기 좋게 설계하고, 학부모가 검색하고 비교하고 문의하기까지의 흐름을 정리합니다."
- **히어로 CTA 신설**: `첫인상 진단 문의하기` (기존 `무료 상담 신청` 대체)
- **S2 헤드**: "제작보다 중요한 건 배치, 배치보다 중요한 건 연결입니다."
- **S9 헤드**: "우리 학원은 지금, 어떻게 보이고 있을까요?"

### 1-4. Basic 재정의 (부제 유지, 요소 5개로 재편)
> **학원 운영을 받쳐주는 기본 마케팅 패키지**

| 제공 요소 | 내용 |
|---|---|
| **첫인상 진단 리포트** ⭐신규 | 플레이스·블로그·인스타·검색 결과 학부모 노출 분석 + 지역 경쟁 학원 비교 + 다음 실행안 |
| 기본 홍보 자산 세트 | 학원 소개 / 강사 / 커리큘럼 / 성과·후기 사례 |
| 플레이스 첫인상 세팅 | 대표 이미지 · 소개글 · 소식 배너 · 적용 화면 목업 |
| 시즌 홍보 기본 세트 | 중간·기말 / 방학특강 / 개강 / 설명회 |
| 10to10 연계 안내 | 설명회 · 레벨테스트 · 학부모 문의 연결 안내 |

**⚠ 갭**: 현재 `월간 운영 리포트` 요소 → 노션안에는 **없음** (Premium 전담 매니저로 이관되는 뉘앙스). 확인 필요.

### 1-5. Premium 재정의 (Basic 전체 + 3요소)
> **디자인·콘텐츠·광고까지 전담 매니저 1:1 케어**

| 제공 요소 | 내용 |
|---|---|
| Basic 전체 포함 | 5요소 그대로 |
| **월간 콘텐츠 운영** | 릴스·숏츠 / 카드뉴스 / 소식글 / 월간 콘텐츠 캘린더 |
| **디지털 광고 운영** | 네이버·구글 키워드 광고 · 소재 기획 · 성과 리뷰 · 다음 액션 조정 |
| **운영 관리** | 월 1회 결과 리뷰 미팅 · 긴급 이슈 대응 · 시즌 홍보 일정 조율 · 콘텐츠 수정 방향 제안 |

**변경 포인트**
- 현재 `숏폼 콘텐츠` → `월간 콘텐츠 운영`으로 확장
- 현재 `전담 매니저` → `운영 관리`로 이름 정리
- 광고비 별도 안내 문구 명시 필요

### 1-6. S6 추가 옵션 (3개로 축소)
- **검색형 블로그 운영팩** ← 신규 블로그 팩 3종 진입점
- **시즌 모집팩** ← 신설 (방학특강/내신/설명회 시즌 홍보물 일괄 교체)
- **현장 홍보물 세트** ← 기존 `offline-print` 재정리

### 1-7. S5 필요 목업 (예솜님 우선순위)
1. 기본 홍보 자산 세트 모음
2. 채널별 재활용 목업
3. 네이버 플레이스 첫 화면 목업 ✅ 보유
4. 블로그 검색 결과 목업
5. 인스타 피드 그리드 목업
6. 시즌 캠페인 세트 목업
7. 첫인상 진단 리포트 내부 페이지 목업
8. 10to10 문의 흐름 목업

### 2-2. 데이터 스키마 갭 (`lib/schema/inquiry.ts`)

```ts
// 현재
package: z.enum(["basic", "premium", "info_session", "blog_build", "custom"])

// 노션안 반영 필요
package: z.enum([
  "basic", "premium", "info_session",
  "blog_setup",      // 첫달 세팅팩 55만
```
